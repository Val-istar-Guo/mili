## Install

```bash
npm install mili -D
```

## Usage

### mili init [option] [app_name]

initial your project

|  option  |      enum      | description |
|:--------:|----------------|-------------|
| -t -type | web, component | Which app type to build?


### mili upgrade

upgrade your project
